# Block

Block is a data structure/collection that uses Blockchain principle in managing data.

- 🚀 Zero dependencies
- 🔌 Less API
- 🛠️ Full TypeScript type inference

## Installation

```bash
npm i block
```

## Usage

### Basic Class Example
```typescript

import { Block, BlockType, block }  from "@block";

```
